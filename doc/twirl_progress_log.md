- `2026-05-06`: First-pass design choices, all forced by what the WD 1856 smoke test exposed:
    - Search **all three apertures** (`DET_FLUX_SML`, `DET_FLUX`, `DET_FLUX_LAG`). Aperture-of-record varies per CCD/contamination — S56 cam4/ccd1 passes via `DET_FLUX`, S59 cam4/ccd4 only passes via `DET_FLUX_SML` (medium and large are scattered-light dominated on that CCD).

    - `n_periods = 200,000`, `n_peaks = 20`. Compute is cheap; keeping more peaks lets the cross-aperture step rescue weak signals that aren't a single aperture's rank-1.
